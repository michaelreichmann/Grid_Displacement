# RGB Pixelated Displacement

*Max 8.3*

This Jitter Effect displaces a texture with a noise texture within a grid.<br/> 
The controls are:<br/>
**dimensions** of the grid<br/>
**amplitude** of the effect<br/>
**button**: creates a new displacement noise<br/>
